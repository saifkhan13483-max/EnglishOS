PROBLEM WITH CURRENT PALETTE:
- #0A0A0F is nearly pure black → causes harsh contrast with white text and OLED smearing.
- Red accent #E94560 is strong but used in isolation — no warm/cool balance system.
- #C8C8E0 lavender-grey secondary text has a purple tint that clashes subtly with warm red.
- Too many brand colors (5) with no clear hierarchy — creates visual noise.
- Borders are too similar to backgrounds — cards don't "lift" off the page.

NEW BACKGROUND SYSTEM (warmer dark, less "void"):
- bg-primary:    #0C0C14    ← (slightly lifted from pure black, adds depth)
- bg-elevated:   #13131E    ← (cards, modals, dropdowns)
- bg-surface:    #1C1C2A    ← (input fields, code blocks, nested surfaces)
- bg-hover:      #242438    ← (hover states, active selections)
- bg-highlight:  #2A2A42    ← (selected items, active tabs)

WHY: Moving from #0A0A0F to #0C0C14 avoids the "OLED void" effect. Each step is a consistent
+8-10% lightness jump, creating a clear 5-level elevation system for depth and hierarchy.

NEW BRAND COLOR SYSTEM (tighter hierarchy):

Primary Accent — KEEP the red but refine it:
- brand-500:  #EF4565    ← (slightly warmer and more saturated than #E94560)
- brand-400:  #F76B84    ← (hover state, lighter variant)
- brand-600:  #D93251    ← (pressed/active state)
- brand-900:  #2A0A12    ← (red-tinted background for alerts/emphasis)
- brand-glow: rgba(239, 69, 101, 0.15)  ← (subtle glow behind CTAs)

Secondary Accent — Elevate blue to a proper secondary:
- secondary-500: #5BA4FF   ← (brighter, more visible than #4A9EFF on dark bg)
- secondary-400: #7DB8FF   ← (hover)
- secondary-600: #3D8AEF   ← (pressed)
- secondary-900: #0A1628   ← (blue-tinted background for info sections)

Semantic Colors (reduce from 5 to 3 + red/blue):
- success:  #34D884     ← (warmer green, higher saturation for dark bg visibility)
- warning:  #F5B731     ← (slightly warmer gold, dual-purpose: warnings + XP/achievements)
- error:    #EF4565     ← (reuse brand-500 — red IS the error color, reduces palette bloat)

REMOVE purple (#A855F7) as a standalone brand color. If needed for rare decorative use,
derive it from a gradient between brand-red and secondary-blue:
- gradient-accent: linear-gradient(135deg, #EF4565, #5BA4FF)  ← replaces purple

NEW TEXT COLORS:
